# SEGURIDAD CORPORATIVA (sitio estático)

Este repositorio contiene el manual online para consulta del personal de seguridad.

## Estructura
- `index.html` inicio
- `manual.html` manual completo
- `anexos.html` listado de anexos
- `videos.html` enlaces a búsquedas de capacitaciones
- `biblioteca.html` recursos y enlaces
- `cambios.html` control de cambios
- `descargas.html` descargas e instrucciones
- `assets/` estilos y JavaScript
- `docs/` colocar PDFs (manual y anexos)

## Publicación en GitHub Pages
1. Subí todo al repositorio.
2. GitHub → Settings → Pages
3. Source: Deploy from a branch
4. Branch: `main` / Folder: `/ (root)`

## Cargar el PDF oficial
Colocá el manual en `docs/` y llamalo:
`docs/MANUALGuardia-de-Seguridad.pdf`

Luego el link de "Descargas" quedará activo.
