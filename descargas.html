Instrucciones rápidas para GitHub Pages

  1. Creá un repositorio (p. ej. seguridad-corporativa).
  2. Subí el contenido de este ZIP tal cual.
  3. En GitHub: Settings → Pages y elegí Deploy from a branch, branch main, carpeta / (root).
  4. Esperá a que se publique el link. Listo: sitio online.
